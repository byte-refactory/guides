# How to Create a GitHub Account

GitHub is a online platform for storing and managing source code. Byte Refactory uses GitHub to securely deliver your
software assets.

If you do not already have a GitHub account, please create one so we can provide access to your project.

> [!TIP]
>
> **Is GitHub Secure?**
>
> GitHub is a widely used Sass platform owned by Microsoft since 2018. It has a strong security track record and undergoes regular
> security and compliance audits.
>
> Your repository will be private, and only authorized users will have access.

## Steps

1. Visit [github.com](https://github.com) and select the **Sign Up** option in the top-right corner.

2. Complete the sign up process.

3. Once logged in, click on your **profile icon** (top-right) and select **Profile**. You're account ID can be found in the URL
   and near the top of the page.

4. Send your account ID to info@byterefactory.com. We'll take care of the rest.
